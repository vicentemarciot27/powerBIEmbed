Teste de Embed de Power BI Ita Júnior

Esse teste é desenvolvido usando NodeJS
Para mudar o relatório mostrado, basta modificar config.json

O Código está presente nos seguintes arquivos:

  1. embedConfigService.js
  2. authentication.js
  3. server.js
  4. index.js
  5. utils.js
  6. index.html
  7. index.css

Desenvolvido por Ita Júnior com base nos templates fornecidos pela Microsoft em https://learn.microsoft.com
